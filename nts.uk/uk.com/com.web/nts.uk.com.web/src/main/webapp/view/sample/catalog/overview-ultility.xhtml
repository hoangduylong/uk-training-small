Overview Utility

What is a utility?

Utilities is available functions that often used to serve for a special task. Because it is often used by everyone, it is created for Consistent and easy for maintenance

Utilities created by Kiban team and still in process. If you have any requirement for utilities, please contact Kiban team. If you change or fix it on your own it'll break Consistent

List of main utility modules

  • Common utility: utilities not in below list.
  • Text utility: utilities for work with string, text
  • Number utility: utilities for work with number
  • Time utility: utilities for work with time, date
  • Request utility: utilities for work request to server
