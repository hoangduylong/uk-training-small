Fixed-Flex Layout

Fixed-Flex Layout is kind of Two Columns Layout.

For example: CMM001 共通会社情報の登録

Usage

You just put the 3 classes: "fixed-flex-layout", "fixed-flex-layout-left", and "fixed-flex-layout-right".
]]>

The width of left column is fixed, the value is decided by own contents if you don't specify it.

The width of right column is flex, will be expanded automatically. It is not needed to specify width.
