Overview Function Wrap

What is a function wrap?

A function wrap is a common control which be used often. But it usually be called in code instead of rendered in HTML. Not like a Component which has HTML binding + ViewModel, Function Wrap only need to call javascript function to use.

List of components

  • SubWindow: open dialog with an embed <iframe>
  • Dialog: open dialog with message only
  • Popup: show a popup with content inside
  • Errors: allow you push and clear error for control
  • DirtyChecker: check if user make any change to a specify value
