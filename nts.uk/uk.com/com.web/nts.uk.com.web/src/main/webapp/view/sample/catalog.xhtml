]]>

We'll use a master, which delare common javascipt and css files for us (and do some more but I won't say it here, talk with Kiban team if you need any other information).

  • base.xhtml: if your page is a page or subwindow
  • dialog.xhtml: if your page is a subwindow only

Header

Whatever you defined inside the following code will get in head


	
	
		nts.uk.shr.com.primitive.PersonId
	
	
	
	
	
]]>
  • <com:ValidatorScript>: auto create constraint base on your primitiveValue. This will be used to validate your input
  • <com:stylefile path="abc.css"/>: use this to add CSS file to your page. DON'T use <link rel="stylesheet" type="text/css" href="abc.css" />
  • <com:scriptfile path="abc.js"/>: use this to add JS file to your page. DON'T use <script src="abc.js" />
