File tags

In this group have custom tags for get js and css files for import into master page.

scriptfile

To import a Javascript file into page. Use attribute 'path' for import a specific javascript file. You can use 'set' attribute for import a set of defined javascript files. Currently, there are two set are 'IGNITE' and 'BASIC'.

stylefile

To import a css file into page. Use attribute 'path' for import a specific javascript file. You can use 'set' attribute for import a set of defined javascript files. Currently, there is only a set 'BASIC'.
