Binding

Description
checked Type: any;
Get or set value of checked RadioBox
checkedValue
(optional, not apply with ntsRadioButton)		 Type: any
Set value for RadioBox
option
(optional)		 Type: any
Set option for RadioBox
enable
(optional)		 Type: boolean
Set enable for RadioBox
name
(optional)		 Type: string
Set data name for RadioBox
optionValue
(optional)		 Type: string
Set value for RadioBox
optionText
(required)		 Type: string
Set text (if option is not defined, use optionText, if option is defined, use optionText property from option) for RadioBox
group
(optional)		 Type: string
Set group for RadioBox
