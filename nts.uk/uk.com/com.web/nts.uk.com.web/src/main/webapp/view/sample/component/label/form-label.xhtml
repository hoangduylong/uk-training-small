API

Binding

Description
text
(optional)		 Type: KnockoutObservable<string> | string
Text for formlabel
constraint
(optional)		 Type: string
Validate constraint for editor. Error message will automatic display by this constraint
inline
(optional)		 Type: boolean
Default: true
Set Label display in 1 line
required
(optional)		 Type: boolean
Default: true
Define that field is required. Change color of Label to orange
enable
(optional)		 Type: boolean
Default: false
Define that field is disable or enable. Change text of Label to grey

Note: (Change constraint in runtime)

Using constraint as observable and call valueHasMutated function after change primitive value.

changePrimitive() {
    let self = this;
    __viewContext.primitiveValueConstraints['ResidenceCode'].maxLength = Math.floor(Math.random() * 20) + 1;

    self.constraint.valueHasMutated();
    self.constraints.valueHasMutated();
}
