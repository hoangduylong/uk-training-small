Limited label

"limited-label" class

We limit length of text that user can input, but we can not control "physical length" of shown texts on screen. For example:

1234567890
WWWWWWWWWW

So when you write code to show texts that is input by user, use "limited-label": 


WWWWWWWWWW




    WWWWWWWWWW

]]>
Demo limited-label
1234567890
WWWWWWWWWW
If text is limited, all text is shown when mouse over.
If the texts is wrapped by any frames:
1234567890
WWWWWWWWWW

Actually ".limited-label" has default CSS: "width: 100%".
