Number ulitily

In this module have functions for work with number in javascript. Below is main functions of it.

isNumber

This function will check a value is number? Default check integer, you can set check float. And pass option is NumberEditorOption. To use Option, please import /lib/nittsu/ui/textbox-options.js

trunc

get integer part of decimal number

getDecimal

get decimal with number scale

formatNumber

format a number with NumberEditorOption
