Overview custom tag

What is a Tag?

An html tag is a signal to a browser that it should do something other than just throw text up on the screen. By convention all HTML tags begin with an open angle bracket (<) and end with a close angle bracket (>). HTML tags are classified into two types: paired tags and singular tags

What is a Custom Tag?

In JSF, the custom tags are user defined sets of the JSP element present in the JSP libraries.The custom tags used in the JSP page are translated into a servlet. The tag is also converted to operations on an object named the tag handler. The tag handler is responsible for the execution of the servlet generated by the translation of the custom tags.

Features of Custom Tag

  • They can be customized via attributes passed from the calling page.
  • They have access to all the objects avaiable to JSP pages.
  • They can modify the response generated by the calling page.
  • They can communicate with each other. You can create and initialize a JavaBeans component. Create a variable that refers to that bean in one tag and then use the Bean in another tag.
  • They can be nested within one another, allowing for complex interactions within a JSP page.
