Binding

Description
image (optional) Type: string
Path to tooltip image
textId (optional) Type: string
ID of text resource to show instead of image
textParams (optional) Type: string
textParams for a ID of text resource to show instead of image
popUpId (optional) Type: string
ID of DIV tag which will be inserted to the help-button content
position Type: string
Set position for popover. See below demo for detail options
enable
(optional)		 Type: boolean
Default: true
Set enable/disable for help button
